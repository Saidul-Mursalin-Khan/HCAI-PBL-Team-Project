See which measurements matter

Choose one measurement. The charts show how the model’s answer tends to change when that measurement becomes smaller or larger.

AccuracySimplicity
Now showing: {{ selected_feature_label }}{{ effect_note }}

How to read these charts

  1. Read from left (smaller measurement) to right (larger measurement).
  2. Each colored line represents one penguin species.
  3. A rising line means the model becomes more likely to choose that species.
{% include "project2/plot.html" with plot=pdp x_label=selected_feature_label %}{% include "project2/plot.html" with plot=ale x_label=selected_feature_label %}
What are these chart methods called?

The first is a Partial Dependence Plot (PDP). The second is an Accumulated Local Effects (ALE) plot. ALE can be safer when measurements influence each other.
