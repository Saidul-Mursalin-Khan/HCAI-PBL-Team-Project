Feature distributions
Histograms
Think of this like a bar chart of "how common" each value is. Each bar shows how many rows in your data have a similar value. Tall bars mean "lots of your data looks like this," short bars mean "this value is rare." If most bars pile up on one side, your data may be lopsided (that's okay, just good to know!).
{% endif %}
{% if plots.boxplot %}
Box Plot
This shows the "typical range" for each number. The box covers where most of your values sit, the line in the middle is the halfway point, and any dots far away from the box are unusual values ("outliers") — like one very tall person in a room full of average-height people.
{% endif %} {% if plots.scatter %}
Scatter Plot
Each dot is one row of your data. This chart helps show whether two things tend to move together — for example, if dots trend upward from left to right, it means as one number goes up, so does the other.
{% endif %} {% if plots.heatmap %}
Correlation Heatmap
This grid shows which columns "move together." Darker or more intense colored squares mean two columns are strongly linked — when one changes, the other tends to change too. Lighter squares mean the two columns don't really affect each other. No need to memorize the numbers — just look for the boldest colors.
{% endif %}
